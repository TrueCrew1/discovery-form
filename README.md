# TrueCrew — Operations Discovery Form

A focused, multi-step diagnostic intake form that identifies workflow gaps in field service and maintenance operations. Prospects complete an 8-step questionnaire (~6 minutes), and TrueCrew responds within 1 business day with a structured gap report and solution path recommendation.

Hosted as a static site on GitHub Pages. Form submissions are handled via Formspree — no backend required.

---

## What This Does

- Walks prospects through 8 diagnostic steps covering operations type, workflow breakdowns, cost impact, current tooling, and readiness to act
- Scores answers and surfaces a preliminary gap profile before final submission
- Sends a structured lead payload to Formspree including full answers and contact info
- Delivers a clear recommendation: custom build, SaaS bundle, or phased rollout
